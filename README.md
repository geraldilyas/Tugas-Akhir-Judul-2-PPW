Pada projek kali ini saya mencoba untuk mengupload hasil kerja saya ke GitHub untuk dapat dengan mudah diakses oleh orang lain yang ingin melihat source 
code saya, langkah yang saya lakukan dalam mengupload source code ini adalah:
1. inisialisasi Git repository dengan perintah "git init"
2. menambahkan file index.html yang ke dalam repo dengan perintah "git add index.html"
3. Commit perubahan yang dilakukan dengan perintah "git commit -m "_pesan commit"_
<img width="512" height="206" alt="image" src="https://github.com/user-attachments/assets/0aece674-7b9e-4938-a190-abc09f3c3e1a" />

4. Menambahkan file style.css dengan perintah "git add style.css"
5. Membuat branch baru untuk bereksperimen dengan styling dari halaman dengan nama "new-style"
6. Menambahkan file style.css yang sudah diperbarui pada branch "new-style"
7. Merge branch utama dengan branch eksperimen 
<img width="409" height="189" alt="image" src="https://github.com/user-attachments/assets/fedbc7a7-8c79-4b05-a7b3-d0b0e82dce21" />

8. Menghubungkan repository lokal ke gitHub
9. Mengupload file lokal ke gitHub dengan perintah "git push"
